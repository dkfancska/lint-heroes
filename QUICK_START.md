## Installation

### macOS/Linux
```bash
# Clone the repository
git clone https://github.com/dkfancska/lint-heroes.git
cd lint-heroes

# Run installation
./macos/installall.sh

# Use aliases
lint-heroes your_file.py
```

### Windows
```powershell
# Clone the repository
git clone https://github.com/dkfancska/lint-heroes.git
cd lint-heroes

# Run installation (PowerShell recommended)
.\windows\installall.ps1

# Use aliases
lint-heroes your_file.py
```
